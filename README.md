# Library Management System (Java)

## 📚 Project Description

This is a simple **Library Management System** developed using **Java**.
The project demonstrates basic Object-Oriented Programming concepts such as classes, objects, encapsulation, and modular design.

The system allows management of books and library members.

## 🚀 Features

* Add new books to the library
* Register library members
* View available books
* Borrow books
* Return books
* Basic library operations handled through the main program

## 🛠️ Technologies Used

* Java (JDK 21)
* Eclipse IDE
* Object-Oriented Programming (OOP)

## 📂 Project Structure

```
src/
 └── com.library
      ├── Book.java
      ├── Member.java
      ├── LibrarySystem.java
      └── Main.java
module-info.java
```

### File Description

* **Book.java** – Represents book details like title, author, and availability.
* **Member.java** – Stores library member information.
* **LibrarySystem.java** – Contains core logic for managing books and members.
* **Main.java** – Entry point of the application where the program runs.

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/your-username/LibraryManagementZOHO.git
```

2. Open the project in Eclipse or any Java IDE.

3. Run the `Main.java` file.
